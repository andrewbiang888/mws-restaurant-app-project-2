## Setup
Here are the steps to startup the database server and python server.

Start with downloading the repo and unziping it. Then navigate (cd) to the folder.

```bash
npm run intall-all
```
if this fails try:
```bash
npm i
```
```bash
cd server
npm i
```
Then run 
```bash 
npm run server
```
